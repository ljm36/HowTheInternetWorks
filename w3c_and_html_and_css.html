W3C and HTML and CSS

W3C means The World Wide Web Consortium.The World Wide Web Consortium exists to realize the full potential of the Web. The W3C is an industry consortium which seeks to promote standards for the evolution of the Web and interoperability between WWW products by producing specifications and reference software. Although W3C is funded by industrial members, it is vendor-neutral,and its products are freely available to all.

HTML means Hypertext Markup Language while CSS refers to Cascading Style Sheet. They are two of the core technologies for building web pages. HTML is the language for describing structures of webpages. With this language, web developers can publish online documents with heading, text, pics, etc. They can retrieve online information via hypertext links, design forms for conduction transactions with remote services, and do many more things. While CSS is the language for describing the presentation of webpages, including colors, layout and fonts. It allows each web developer to express themselves in a way and change the website to their liking and making it their own.
